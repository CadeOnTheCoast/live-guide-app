# Project Roadmap

This document outlines the planned features, historical milestones, and future direction of the Live Guide App.

## Phase 1: Foundation (Completed)
- [x] Initial project setup (Next.js, Prisma, Tailwind).
- [x] Supabase Magic Link authentication integration.
- [x] Baseline database schema (People, Projects, Objectives, Milestones).
- [x] Admin dashboard for basic project management.

## Phase 2: Data & Automation (Completed)
- [x] Automated data fetching pipeline (StoneEagle reports).
- [x] GitHub Actions for daily data synchronization.
- [x] Cloud deployment to Vercel with automated CI/CD.
- [x] CSV/Excel export functionality for dashboard data.

## Phase 3: Enhanced UI/UX (In Progress / Completed)
- [x] Interactive Master Timeline and Project Timeline views.
- [x] Improved tooltip readability and chart interactions.
- [x] Modern, responsive design system (vibrant colors, glassmorphism).
- [x] Dynamic project overview forms with hydration fixes.

## Phase 4: Budgeting & Financials (Completed)
- [x] Unified budget data ingestion (merging raw & narrative sources).
- [x] Detailed Budget Table with collapsible notes and recurrent expenses.
- [x] Project financial summary implementation.

## Phase 5: Intelligence & Analytics (Upcoming)
- [/] **Predictive Analytics Engine**: Time series forecasting for key metrics (VSC%, GAP%, PVR).
- [ ] **Anomaly Detection**: Highlight stores or projects with significant deviations from trends.
- [ ] **Goal Probability**: Calculate the likelihood of achieving project milestones based on current velocity.
- [ ] **Advanced Filtering**: Granular date ranges and comparisons across stores.

## Future Horizons
- [ ] **GIS Integration**: Fully resolve SSL/Certificate issues for real-time GIS data pulling.
- [ ] **Enhanced Admin Tools**: More robust role-based access control (RBAC) and audit logs.
- [ ] **Mobile Optimization**: Dedicated mobile view for stakeholders on the go.
- [ ] **Custom Notifications**: Slack/Email alerts for significant project events or missed milestones.

---
> [!NOTE]
> This roadmap is living and subject to change based on stakeholder feedback and evolving project requirements.
